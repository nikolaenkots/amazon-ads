Правила изменения ставок развернуть
Правила хранятся в базе и применяются на сервере. Два независимых набора: Оптимизация — по кликам ключа и ACOS (при достаточной статистике), Разгон — по показам (только для таргетов с малым числом показов). Наборы не перекрывают друг друга: оптимизация имеет приоритет, разгон срабатывает когда данных мало. Приоритет области: портфолио+таргетинг → портфолио → глобальное+таргетинг → глобальное.
